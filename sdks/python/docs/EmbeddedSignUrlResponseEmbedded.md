## Properties

| Name | Type | Description | Notes |
| ---- | ---- | ----------- | ----- |
| `sign_url` | ```str``` |  A signature url that can be opened in an iFrame.  |  |
| `expires_at` | ```int``` |  The specific time that the the `sign_url` link expires, in epoch.  |  |
